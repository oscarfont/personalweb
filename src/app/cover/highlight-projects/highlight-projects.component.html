DailyWordBot

An open source Telegram chatbot to help you learn Spanish and German vocabulary. The bot sends three words a day with an example.

My Contribution to the project was to add the feature that lets the user decide the level of the words that wishes to receive. Words are labeled with the levels: beginner, intermediate and advanced. Check the code.

Pro Wrestling Sentiment

Web application that uses the Social Media Sentiment API. Analysed 70k tweets and 30k Instagram posts related to WWE superstars. This was also part of my Master thesis

The frontend is a dashboard that represents the most interesting stats of the analyzed posts and has a ranking of shows or wrestlers. The serverless backend deployed at AWS stores in DynamoDB the most popular post and the mentioned stats. More details.

Social Media Sentiment API

Microservice API used to look for social media posts and tweets and return them with its related sentiment (positive or negative).

This microservice backend consists in a Nodejs Api gateway that exposes to the Internet two internal microservices, one in Java, that crawls social media, and another one in Python that conducts the sentiment analysis of the posts. This was part of my Master thesis. More details.

Dixie Virtual Assistant

Android application developed with Java, as part of my Bachelor final thesis to learn how Virtual Assistants work.

Dixie lets you publish tweets and schedule posts for Instagram by voice. The application uses Google Speech to text service for the voice commands transcription. It uses wit.ai to extract the intent of the commands and stores the chat messages locally in an SQLite database. More details
